# projeto-final-ana-almeida
Projeto final da UC - Laboratório de Projeto II

## Descrição
Para a realização do Projeto final da UC de Laboratório de Projeto II, irei realizar um website pessoal. Nesse website irei colocar os trabalhos desenvolvidos nas duas licenciaturas que tenho, design de espaços e design gráfico e multimédia.

---

## Objetivos
- Desenvolver um website pessoal com o meu portfólio;
- Organizar e apresentar os meus trabalhos académicos;
- Aplicar conhecimentos de HTML, CSS e JavaScript;
- Integrar projetos desenvolvidos em Processing.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Processing

---

## Páginas do Website
- **index.html** -> Página Principal
- **sobre-mim.html** -> As minhas informações
- **projetos.html** -> Apresentação dos vários trabalhos desenvolvidos
- **contactos.html** -> Os meus contactos
- **processing.html** -> Projetos desenvolvidos em Processing (com canvas)
